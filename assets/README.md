# assets/

Metti qui la roba vera (ora placeholder generati via CSS):

- `photos/` — foto workshop/archivio (trattate come poster strappati/collage)
- `logos/` — loghi partner per la sezione Collaborazioni
- `fonts/` — font veri (stencil/marker/condensed + brutalist serif)

Poi aggancia i file da `css/style.css` (variabili `--font-*`, `.poster__img`,
`.partner`) e da `data/courses.js` se vuoi passare percorsi immagine nei record.
